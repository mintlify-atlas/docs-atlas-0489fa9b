title: "Accounts"
description: "Manage bank accounts for authenticated users"

<Warning>
  All endpoints on this page require a valid `Authorization: Bearer <token>` header. Requests without a valid token return `401 Unauthorized`.
</Warning>

## List accounts

<br />

`GET /api/ledger/accounts`

Returns all bank accounts belonging to the authenticated user.

### Response

<ResponseField name="id" type="string" required>
  Unique account identifier (UUID).
</ResponseField>

<ResponseField name="accountNumber" type="string" required>
  The account number (CLABE or other identifier).
</ResponseField>

<ResponseField name="accountType" type="string" required>
  Type of account, e.g., `"DEFAULT"`, `"checking"`, `"savings"`.
</ResponseField>

<ResponseField name="balance" type="number" required>
  Current account balance.
</ResponseField>

<ResponseField name="currency" type="string" required>
  ISO 4217 currency code, e.g., `"MXN"`.
</ResponseField>

<CodeGroup>

```bash curl
curl --request GET \
  --url http://localhost:5000/api/ledger/accounts \
  --header 'Authorization: Bearer <token>'
```

</CodeGroup>

```json Response
[
  {
    "id": "3fa85f64-5717-4562-b3fc-2c963f66afa6",
    "accountNumber": "646180123456789012",
    "accountType": "DEFAULT",
    "balance": 15000.00,
    "currency": "MXN"
  }
]
```

## Create account

<br />

`POST /api/ledger/accounts`

Creates a new bank account for the authenticated user. Returns the created account object with a `201 Created` status.

### Request body

<ParamField body="accountNumber" type="string" required>
  Account number for the new account. Must be between 10 and 50 characters.
</ParamField>

<ParamField body="accountType" type="string" required>
  Type of account to create, e.g., `"DEFAULT"`, `"checking"`, `"savings"`. Must be between 3 and 50 characters.
</ParamField>

### Response

Returns the newly created account object. See the [List accounts](#list-accounts) response fields for the full schema.

### Error responses

| Status | Description |
|--------|-------------|
| `400`  | Validation error — missing or invalid fields. |
| `401`  | Invalid or missing bearer token. |

<CodeGroup>

```bash curl
curl --request POST \
  --url http://localhost:5000/api/ledger/accounts \
  --header 'Authorization: Bearer <token>' \
  --header 'Content-Type: application/json' \
  --data '{
  "accountNumber": "646180123456789012",
  "accountType": "DEFAULT"
}'
```

</CodeGroup>

```json Response (201 Created)
{
  "id": "3fa85f64-5717-4562-b3fc-2c963f66afa6",
  "accountNumber": "646180123456789012",
  "accountType": "DEFAULT",
  "balance": 0.00,
  "currency": "MXN"
}
```
