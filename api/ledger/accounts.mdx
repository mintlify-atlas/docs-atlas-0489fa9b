<Warning>
  All endpoints on this page require a valid `Authorization: Bearer <token>` header. Requests without a valid token return `401 Unauthorized`.
</Warning>

## List accounts

<br />

`GET /api/ledger/accounts`

Returns all bank accounts belonging to the authenticated user.

### Response

<ResponseField name="id" type="string" required>
  Unique account identifier (UUID).
</ResponseField>

<ResponseField name="userId" type="string" required>
  UUID of the account owner.
</ResponseField>

<ResponseField name="accountType" type="string" required>
  Type of account, e.g., `"checking"` or `"savings"`.
</ResponseField>

<ResponseField name="currency" type="string" required>
  ISO 4217 currency code, e.g., `"MXN"`.
</ResponseField>

<ResponseField name="alias" type="string">
  Friendly name assigned to the account.
</ResponseField>

<ResponseField name="clabe" type="string" required>
  18-digit CLABE interbank number assigned to the account.
</ResponseField>

<ResponseField name="balance" type="number" required>
  Current account balance.
</ResponseField>

<ResponseField name="createdAt" type="string" required>
  ISO 8601 timestamp of account creation.
</ResponseField>

<CodeGroup>

```bash curl
curl --request GET \
  --url http://localhost:5000/api/ledger/accounts \
  --header 'Authorization: Bearer <token>'
```

</CodeGroup>

```json Response
[
  {
    "id": "3fa85f64-5717-4562-b3fc-2c963f66afa6",
    "userId": "user-uuid",
    "accountType": "checking",
    "currency": "MXN",
    "alias": "Cuenta principal",
    "clabe": "646180123456789012",
    "balance": 15000.00,
    "createdAt": "2026-03-01T10:00:00Z"
  }
]
```

## Create account

<br />

`POST /api/ledger/accounts`

Creates a new bank account for the authenticated user. Returns the created account object with a `201 Created` status.

### Request body

<ParamField body="accountType" type="string" required>
  Type of account to create, e.g., `"checking"` or `"savings"`.
</ParamField>

<ParamField body="currency" type="string" required>
  ISO 4217 currency code for the account, e.g., `"MXN"`.
</ParamField>

<ParamField body="alias" type="string">
  Optional friendly name for the account.
</ParamField>

### Response

Returns the newly created account object. See the [List accounts](#list-accounts) response fields for the full schema.

### Error responses

| Status | Description |
|--------|-------------|
| `400`  | Validation error — missing or invalid fields. |
| `401`  | Invalid or missing bearer token. |

<CodeGroup>

```bash curl
curl --request POST \
  --url http://localhost:5000/api/ledger/accounts \
  --header 'Authorization: Bearer <token>' \
  --header 'Content-Type: application/json' \
  --data '{
  "accountType": "checking",
  "currency": "MXN",
  "alias": "Cuenta principal"
}'
```

</CodeGroup>

```json Response (201 Created)
{
  "id": "3fa85f64-5717-4562-b3fc-2c963f66afa6",
  "userId": "user-uuid",
  "accountType": "checking",
  "currency": "MXN",
  "alias": "Cuenta principal",
  "clabe": "646180123456789012",
  "balance": 0.00,
  "createdAt": "2026-03-17T14:00:00Z"
}
```
