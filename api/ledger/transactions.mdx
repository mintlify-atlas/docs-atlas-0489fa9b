<Warning>
  All endpoints on this page require a valid `Authorization: Bearer <token>` header. Requests without a valid token return `401 Unauthorized`.
</Warning>

<Info>
  Before submitting a transaction, the frontend evaluates the transfer with [`POST /ai/risk`](/api/ai/risk). If the returned risk level is `medium` or `high`, the user is shown a `ConfirmDialog` before the transfer is submitted.
</Info>

## List transactions

<br />

`GET /api/ledger/transactions`

Returns all transactions for the authenticated user.

### Response

<ResponseField name="id" type="string" required>
  Unique transaction identifier (UUID).
</ResponseField>

<ResponseField name="fromAccountId" type="string" required>
  UUID of the sender's account.
</ResponseField>

<ResponseField name="toBeneficiaryId" type="string" required>
  UUID of the beneficiary who received the transfer.
</ResponseField>

<ResponseField name="beneficiaryName" type="string" required>
  Full name of the beneficiary at the time of the transaction.
</ResponseField>

<ResponseField name="amount" type="number" required>
  Transfer amount. Must be greater than zero.
</ResponseField>

<ResponseField name="status" type="string" required>
  Current transaction status, e.g., `"completed"`, `"pending"`, `"failed"`.
</ResponseField>

<ResponseField name="speiReference" type="string">
  SPEI tracking reference number, if available.
</ResponseField>

<ResponseField name="createdAt" type="string" required>
  ISO 8601 timestamp of when the transaction was created.
</ResponseField>

<CodeGroup>

```bash curl
curl --request GET \
  --url http://localhost:5000/api/ledger/transactions \
  --header 'Authorization: Bearer <token>'
```

</CodeGroup>

```json Response
[
  {
    "id": "txn-uuid",
    "fromAccountId": "account-uuid",
    "toBeneficiaryId": "ben-uuid",
    "beneficiaryName": "Juan Pérez",
    "amount": 500.00,
    "status": "completed",
    "speiReference": "SPEI20260317001",
    "createdAt": "2026-03-17T14:30:00Z"
  }
]
```

## Create transaction

<br />

`POST /api/ledger/transactions`

Creates a new SPEI transfer from the authenticated user's account to a saved beneficiary.

<Tip>
  Call [`POST /ai/risk`](/api/ai/risk) before this endpoint and present a confirmation dialog to the user when the risk level is `medium` or `high`. The transfer requires a beneficiary to be saved first — use [`POST /api/ledger/beneficiaries`](/api/ledger/beneficiaries) to add one.
</Tip>

### Request body

<ParamField body="fromAccountId" type="string" required>
  UUID of the account to debit.
</ParamField>

<ParamField body="toBeneficiaryId" type="string" required>
  UUID of the saved beneficiary to credit. The beneficiary must already exist — see [Beneficiaries](/api/ledger/beneficiaries).
</ParamField>

<ParamField body="amount" type="number" required>
  Transfer amount. Must be greater than zero (`0.01` minimum).
</ParamField>

### Error responses

| Status | Description |
|--------|-------------|
| `400`  | Insufficient funds, missing fields, or invalid amount. |
| `401`  | Invalid or missing bearer token. |

<CodeGroup>

```bash curl
curl --request POST \
  --url http://localhost:5000/api/ledger/transactions \
  --header 'Authorization: Bearer <token>' \
  --header 'Content-Type: application/json' \
  --data '{
  "fromAccountId": "3fa85f64-5717-4562-b3fc-2c963f66afa6",
  "toBeneficiaryId": "ben-uuid",
  "amount": 500.00
}'
```

</CodeGroup>

```json Response (200 OK)
{
  "id": "txn-uuid",
  "fromAccountId": "3fa85f64-5717-4562-b3fc-2c963f66afa6",
  "toBeneficiaryId": "ben-uuid",
  "beneficiaryName": "Juan Pérez",
  "amount": 500.00,
  "status": "completed",
  "speiReference": "SPEI20260317001",
  "createdAt": "2026-03-17T14:30:00Z"
}
```
