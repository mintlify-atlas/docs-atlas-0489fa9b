<Warning>
  This endpoint requires a valid `Authorization: Bearer <token>` header. Requests without a valid token return `401 Unauthorized`.
</Warning>

## GET /api/openfinance/products

Returns all financial products — accounts, credit cards, and loans — pulled from the user's connected external banks. Products are populated or refreshed by calling [`POST /api/openfinance/sync/{connectionId}`](/api/openfinance/connections#sync-connection).

### Response

<ResponseField name="id" type="string" required>
  Unique product identifier (UUID).
</ResponseField>

<ResponseField name="connectionId" type="string" required>
  UUID of the connection this product belongs to.
</ResponseField>

<ResponseField name="productType" type="string" required>
  Type of financial product, e.g., `"checking"`, `"savings"`, `"credit_card"`, `"loan"`.
</ResponseField>

<ResponseField name="productName" type="string" required>
  Display name of the product as reported by the external institution.
</ResponseField>

<ResponseField name="balance" type="number" required>
  Current balance as of the last sync.
</ResponseField>

<ResponseField name="currency" type="string" required>
  ISO 4217 currency code, e.g., `"MXN"`.
</ResponseField>

<ResponseField name="lastUpdated" type="string" required>
  ISO 8601 timestamp of the last time this product's data was refreshed.
</ResponseField>

<CodeGroup>

```bash curl
curl --request GET \
  --url http://localhost:5000/api/openfinance/products \
  --header 'Authorization: Bearer <token>'
```

</CodeGroup>

```json Response
[
  {
    "id": "prod-uuid",

  }
```
