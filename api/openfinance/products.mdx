<Warning>
  This endpoint requires a valid `Authorization: Bearer <token>` header. Requests without a valid token return `401 Unauthorized`.
</Warning>

## GET /api/openfinance/products

Returns all financial products — accounts, credit cards, and loans — pulled from the user's connected Open Finance providers. Products are populated or refreshed by calling [`POST /api/openfinance/sync/{connectionId}`](/api/openfinance/connections#sync-connection).

### Response

<ResponseField name="id" type="string" required>
  Unique product identifier (UUID).
</ResponseField>

<ResponseField name="connectionId" type="string" required>
  UUID of the connection this product belongs to.
</ResponseField>

<ResponseField name="provider" type="string" required>
  Name of the provider that reported this product, e.g., `"BBVA"`.
</ResponseField>

<ResponseField name="productType" type="string" required>
  Type of financial product, e.g., `"checking"`, `"savings"`, `"credit_card"`, `"loan"`.
</ResponseField>

<ResponseField name="name" type="string" required>
  Display name of the product as reported by the provider.
</ResponseField>

<ResponseField name="balance" type="number" required>
  Current balance as of the last sync.
</ResponseField>

<ResponseField name="currency" type="string" required>
  ISO 4217 currency code, e.g., `"MXN"`.
</ResponseField>

<ResponseField name="nextPaymentAmount" type="number">
  Next scheduled payment amount (applicable to loans or credit cards), if available.
</ResponseField>

<ResponseField name="nextPaymentDate" type="string">
  Date of the next scheduled payment in ISO 8601 date format, if applicable.
</ResponseField>

<ResponseField name="lastSync" type="string" required>
  ISO 8601 timestamp of when this product's data was last refreshed.
</ResponseField>

<CodeGroup>

```bash curl
curl --request GET \
  --url http://localhost:5000/api/openfinance/products \
  --header 'Authorization: Bearer <token>'
```
