---
title: "User Profile"
description: "Get and update the authenticated user's profile"
---

<Warning>
  Both endpoints require a valid JWT Bearer token. See [Login](/api/auth/login) to obtain one.
</Warning>

## Endpoints

| Method | Path | Description |
| --- | --- | --- |
| `GET` | `/api/profile/me` | Retrieve the authenticated user's profile |
| `PUT` | `/api/profile/me` | Update the authenticated user's profile |

---

## GET /api/profile/me

Returns profile data for the currently authenticated user.

### Response

#### 200 OK

<ResponseField name="id" type="string" required>
  UUID of the authenticated user.
</ResponseField>

<ResponseField name="email" type="string" required>
  User's registered email address.
</ResponseField>

<ResponseField name="firstName" type="string" required>
  User's first name.
</ResponseField>

<ResponseField name="lastName" type="string" required>
  User's last name.
</ResponseField>

<ResponseField name="createdAt" type="string" required>
  ISO 8601 timestamp of when the account was created.
</ResponseField>

### Example

<CodeGroup>

```bash cURL
curl -X GET http://localhost:5000/api/profile/me \
  -H "Authorization: Bearer <token>"
```

```json Response
{
  "id": "3fa85f64-5717-4562-b3fc-2c963f66afa6",
  "email": "maria@example.com",
  "firstName": "María",
  "lastName": "González",
  "createdAt": "2026-03-01T10:00:00Z"
}
```

</CodeGroup>

---

## PUT /api/profile/me

Updates the profile of the currently authenticated user. Only the provided fields are updated.

### Request body

<ParamField body="firstName" type="string">
  Updated first name.
</ParamField>

<ParamField body="lastName" type="string">
  Updated last name.
</ParamField>

### Response

#### 200 OK

Returns the updated user profile with the same structure as `GET /api/profile/me`.

#### Error responses

| Status | Description |
| --- | --- |
| `400 Bad Request` | Validation error. |
| `401 Unauthorized` | Invalid or expired token. |

### Example

<CodeGroup>

```bash cURL
curl -X PUT http://localhost:5000/api/profile/me \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{
    "firstName": "María",
    "lastName": "López"
  }'
```

```json Response
{
  "id": "3fa85f64-5717-4562-b3fc-2c963f66afa6",
  "email": "maria@example.com",
  "firstName": "María",
  "lastName": "López",
  "createdAt": "2026-03-01T10:00:00Z"
}
```

</CodeGroup>
